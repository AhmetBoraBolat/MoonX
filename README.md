# MoonX

My project named MoonX takes your date of birth, retrieves your zodiac sign, and provides a horoscope reading. Additionally, it can generate horoscope readings based on different categories. Moreover, it allows you to access information related to your current location. In the meditation section, you can listen to music. Once you start the music, you can control it in a small box throughout the application.
